Fitting logistic regression model

We chose the 6 highest association by predictor variables based on previous data prep because fitting every predictor variable took way too long.

glm_telecom <- glm(
  formula = target ~ OnlineSecurity * TechSupport * InternetService * PaymentMethod * OnlineBackup *
                      DeviceProtection,  
  family = "binomial",
  data = telecom %>%
    dplyr::filter(train_test == "Train") # training on train data
)

# After getting a model now we want predictions
telecom <- telecom %>%
  dplyr::mutate(
    predict_probability = predict(
      object = glm_telecom,
      newdata = telecom,
      type = "response" # response gives us the probability, link function gives us log odds, terms is the third option
    ),
    predict_class = round(predict_probability) # this gives us the classification
  )

Area under Curve of ROC

prediction_Train <- ROCR::prediction(
  predictions = telecom$predict_probability[telecom$train_test == "Train"],
  labels = telecom$target[telecom$train_test == "Train"]
)
ROCR::performance(
  prediction.obj = prediction_Train,
  measure = "auc"
)@y.values
## [[1]]
## [1] 0.8

For the training set the area under the ROC curve is .8.

prediction_Test <- ROCR::prediction(
  predictions = telecom$predict_probability[telecom$train_test == "Test"],
  labels = telecom$target[telecom$train_test == "Test"]
)
ROCR::performance(
  prediction.obj = prediction_Test,
  measure = "auc"
)@y.values
## [[1]]
## [1] 0.79

It makes sense that the training data does a better job than test data discriminating. At .78 for the test data, our model does a decent job discriminating.
