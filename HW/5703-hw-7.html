Fitting multiple linear regression

Here we used stepwise selection to select from the full model will all predictors. Then the selected predictors were used for the multiple regression model.

full_model <- YJ_love ~ brand + category + price + rating + number_of_reviews +
              MarketingFlags + value_price + online_only + exclusive +
              limited_edition + limited_time_offer

lm_sephora2 <- lm(
  formula =  full_model,
  data = sephora %>% dplyr::filter(
    Partition == "Train"  
  )
)
MASS::stepAIC(
  object = lm_sephora2,
  direction = "both",
  k = log(
    x = nrow(sephora)
  ),
  trace = 0
)
